Keyboard Shortcuts

If you see a menu, menu item, or button with an underlined letter in its name, you can select that item by holding down the Alt key and simultaneously pressing the letter that is underlined. For example, open the Claim menu by pressing Alt+C, and then choose the New Claim menu item by pressing Alt+N.

You can sort on a second list column by holding the Shift key and simultaneously clicking the link in the column header.

Alt+` Switch the active window frame between the main frame or the currently visible activity or validation frame.
Alt+Down Display the list of options available in a drop-down list (equivalent to clicking on the list). This is different from pressing Down by itself in that Down selects the next item in the list, while Alt+Down displays the full list of items. Once the list is displayed, you can use the Up and Down arrow keys to move within the list.
