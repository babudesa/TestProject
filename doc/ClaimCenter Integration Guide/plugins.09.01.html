ClaimCenter Integration Guide : Plugin Overview
Plugin Overview
ClaimCenter plugins are software modules that ClaimCenter calls to perform an action or calculate a result. ClaimCenter defines a set of plugin interfaces, most of which are optional. You can write your own implementations of plugins in either Gosu or Java. For the complete list of all ClaimCenter plugins, see Summary of All ClaimCenter Plugins.
Related topics include:
•
For interfaces, see “Interfaces” in the Gosu Reference Guide.
•
For writing Java code, see “Java and Gosu” in the Gosu Reference Guide
IMPORTANT For additional Java information that you need to know to write Java plugins, see “Java and Gosu” in the Gosu Reference Guide.
•
For messaging plugins, see “Messaging and Events”.
•
For authentication plugins, see “Authentication Integration”.
•
For document and form plugins, see “Document Management”.
•
For other plugins, see “Other Plugin Interfaces”.
