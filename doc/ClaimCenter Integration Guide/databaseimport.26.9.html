Staging Table Import of Encrypted Properties
If you are importing records using staging table import, your staging table data must have appropriately encrypted properties before importing them into ClaimCenter. ClaimCenter provides a special tool you can run to encrypt data in staging tables before loading.
For more information about these special encryption-related staging table features, see "Encryption Features for Staging Tables".
