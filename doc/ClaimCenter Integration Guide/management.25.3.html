ClaimCenter Integration Guide : Management Integration : The Abstract Management Plugin Interface
The Abstract Management Plugin Interface
The following tabled lists the important interfaces and classes used by the abstract management plugin interface.
 
Interface or class
Description
ManagementPlugin
The management interface for Guidewire systems. This is the interface that the included JMX plugin implements.
GWMBean
The interface for managed beans exposed by the system. The management plugin must expose these internal management beans to the outside world using JMX, SMNP, or some other management interface.
GWMBeanInfo
Meta-information about a Guidewire managed bean (GWMBean), such as its name and description.
ManagementAuthorizationCallbackHandler
A callback handler interface that ClaimCenter invokes if a user attempts management operations.
NotificationSenderMarker
A marker interface that indicates that a GWMBean can send notifications.
Attribute
The Attribute class encapsulates two strings that represent a system attribute name and its value.
AttributeInfo
Metadata about an Attribute, such as name, description, type, and whether the attribute is readable and/or writable.
Notification
A ClaimCenter notification, such as those sent if ClaimCenter locks out a user due to too many failed login attempts.
NotificationInfo
Metadata about a notification, such as a message string, a sequence number, and a notification type.
The main task of the management plugin is to register GWMBean objects. Registering the object means that the plugin provides that service to the outside world in whatever way makes sense for that service. For example, the management plugin might create a wrapper for the GWMBean and expose it using JMX or SNMP using its own published service.
