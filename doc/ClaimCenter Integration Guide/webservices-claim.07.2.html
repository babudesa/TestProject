ClaimCenter Integration Guide : Claim-related Web Services : Claim APIs
Claim APIs
The claim API (IClaimAPI) interface manipulates claims. Important methods include:
•
addActivityFromPattern - Adds an activity to a claim using an activity pattern as a template.
•
addDocument - Registers a link in the claim file to a document stored in an external document repository. See “Document Management Overview” for more details.
•
addNote - Adds a note to a claim file.
•
addFNOL - Creates a new claim as a first notice of loss (FNOL), the first time an insurance copy is notified about a claim. Note the difference between this and the migrateClaim method. See “Adding First Notice of Loss”.
•
migrateClaim - Loads a claim that is new to ClaimCenter but that is not a new claim (FNOL). See “Migrating a Claim”.
These methods are for active claims only. If an external system posts any new updates on an archived claim, IClaimAPI methods throw the exception EntityStateException.
