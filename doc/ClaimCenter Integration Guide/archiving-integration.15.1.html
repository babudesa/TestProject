ClaimCenter Integration Guide : Archiving Integration
Archiving Integration
ClaimCenter supports archiving a claim as a serialized stream of data. You can choose to store the data in a file, in a document storage system, or in a database (as a single large binary object). The data format is a XML document.
IMPORTANT This topic discusses plugins. Plugins are software modules that ClaimCenter calls to perform an action or calculate a result. For information on plugins, see “Plugin Overview”. For the complete list of all ClaimCenter plugins, see “Summary of All ClaimCenter Plugins”.
