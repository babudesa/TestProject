Typelist Filters
It is possible to configure a typelist so that ClaimCenter filters the typelist values so that they do not all appear in the drop-down list (typelist) in the ClaimCenter interface. Guidewire divides typelist filters into the following categories:
 
Type
Creates...
See...
Static
A fixed (static) subset of the values on a typelist. You can create filters that:
•
Include certain specific typecodes on the typelist only.
•
Include certain specific categories of typecodes on the typelist.
•
Exclude certain specific typecodes from the full list of the typecodes on the typelist
“Static Filters”
Dynamic
A dynamic subset of the values on a typelist. You can create filters that:
•
Associate one or more typecodes on a parent typelist with one or more typecodes on a child typelist.
•
Associate all the typecodes on a parent typelist with one or more typecodes on a child typelist.
“Dynamic Filters”
