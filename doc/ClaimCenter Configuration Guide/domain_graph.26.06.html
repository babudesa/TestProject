Accessing the Domain Graph
Guidewire provides access to the domain graph DOT file through the Server Tools pages that are accessible to those with system administration privileges. For a discussion of the server tools, see “Using the Server Tools” in the System Administration Guide.
The Server Tools → Archive Graph Info page contains two tabs (technically known as cards). They are:
•
Graph—Provides human-readable text versions of the domain and admin graphs. It also contains a Download button that generates a ZIP file that contains the .dot files for the domain and admin graphs.
•
Warnings—Provides a list of graph issues that can potentially lead to errors at application start up. Guidewire strongly recommends that you review this page any time that you make a change to the data model.
To access the domain graph
1.
Log into Guidewire ClaimCenter using an administrative account.
2.
Navigate to Server Tools.
3.
Click Info Pages in the left-hand side menu.
4.
Select Archive Graph Info from the drop-down list at the top of the page.
From this screen, you can:
•
View the human-readable text versions of the domain and admin graphs.
•
Download a ZIP file that contains the .dot files for the domain and admin graphs.
•
View (from the Warnings tab) any warnings generated during validation of these graphs.
