Gosu Reference Guide : Gosu Types : Access to Java Types
Access to Java Types
Gosu is built on top of the Java language. Because Gosu provides an dynamic type system, Gosu includes a built-in typeloader for all Java types. This means that from Gosu you have full direct access to Java types, such as Java classes and Java libraries. Simply access the type directly from Gosu and it works just like in Java. The access to Java types includes the following:
•
instantiate Java objects with the standard Gosu new keyword
•
call static methods on Java classes
•
call object methods on instantiated objects
•
get properties from Java objects
