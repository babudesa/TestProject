Gosu Reference Guide : Annotations and Interceptors
Annotations and Interceptors
Gosu annotations are a simple syntax to provide metadata about a Gosu class, constructor, method or property. This annotation can control the behavior of the class, the documentation for the class. For more advanced usage, annotations can define an interceptor for a method. This allows you to wrap a method with code before the method runs, after the method runs, or both. This enables certain types of aspect programming.
