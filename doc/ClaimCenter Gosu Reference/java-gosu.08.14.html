Gosu Enhancements in Java Entity Libraries
Gosu enhancements export to Java code through the entity libraries. See the “Enhancements” in the Gosu Reference Guide for more information about Gosu enhancements.
Both properties and methods defined in enhancements export to Java. There is no special task necessary to use this feature. Enhanced properties and methods are simply available on Guidewire entities after you regenerate the entity libraries (part of the Java API libraries).
