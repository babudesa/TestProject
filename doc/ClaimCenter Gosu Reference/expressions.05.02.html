Gosu Operators
Gosu uses standard programming operators to perform a wide variety of mathematical, logical, and object manipulation operations. If you are familiar with the C, C++ or Java programming languages, you might find that Gosu operators function similar to those other languages. Gosu evaluates operators within an expression or statement in order of precedence. (For details, see “Operator Precedence”.)
Gosu operators take either a single operand (unary operators), two operands (binary operators), or three operands (a special case ternary operator). The following list provides examples of each operator type:
