Static Property Paths
Gosu uses the dot-separated path rooted at a Type expression to retrieve a static property’s value.
Syntax
<type-expression>.<static-property>
Examples
 
Expression
Result
Claim.TypeInfo
Claim typeInfo
LossCause.TC_HAIL
"hail" typekey
java.util.Calendar.FRIDAY
Friday value
For more information about static properties in classes, see “Modifiers”.
