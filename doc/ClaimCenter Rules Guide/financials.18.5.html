Creating Checks and Recoveries
Guidewire provides methods to create checks and recoveries on the following objects:
•
Claim
•
Exposure
•
CheckSet
Claim object. Guidewire provides the following methods for working with foreign currency calculations on a Claim object. Both of these methods have multiple, overloaded versions.
Claim.createCheck( payee, payeeRole, recipient, mailToAddress, reportabilityType, costType,
        costCategory, paymentType, lineCategory, checkAmount, comments, memo, payTo, scheduledSendDate,
        requestingUser )
  
Claim.createRecovery( payer, costType, costCategory, recoveryCategory, lineCategory, recoveryAmount,
        recoveryCurrency, exchangeRateOverride, customExchangeRateDescription, claimAmountOverride,
        reportingAmountOverride, comments, requestingUser )
Exposure object. Guidewire provides the following methods for working with foreign currency calculations on an Exposure object. Both of these methods have multiple, overloaded versions.
Exposure.createCheck( payee, payeeRole, recipient, mailToAddress, reportabilityType, costType,
        costCategory, lineCategory, checkAmount, comments, memo, payTo, scheduledSendDate,
        requestingUser )
  
Exposure.createRecovery( payer, costType, costCategory, recoveryCategory, lineCategory,
        recoveryAmount, comments, requestingUser )
CheckSet object. Guidewire provides the following method for working with foreign currency calculations on a CheckSet object:
CheckSet.newCheck( exposure, payee, payeeRole, recipient, mailToAddress, reportabilityType, costType,
        costCategory, paymentType, lineCategory, checkAmount, checkCurrency,
        transToClaimExchangeRateOverride, transToClaimExchangeRateDescription, claimAmountOverride,
        reportingAmountOverride, comments, memo, payTo, scheduledSendDate, requestingUser )
