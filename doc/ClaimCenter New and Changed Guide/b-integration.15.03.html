Major New SOAP Features in GScript
The GScript language lets you write web services (SOAP APIs) that external systems can call. Simply add a single line of code before the definition of a GScript class to publish it. You can also now write code that easily imports SOAP APIs published from external systems. By subscribing to the external web service and specifying it in Guidewire Studio, you can easily call external web services from GScript. For details, see “Web Services (SOAP)” in the Integration Guide.
