New Contextual Right-Click Menu Commands
Guidewire has modified the Studio right-click menu (accessed by selecting a resource, then performing a mouse right-click) and added additional new functionality. These new commands primarily add the following functionality:
 
Diff Against Base
You must link Studio to an external “difference” tool (such as Araxis Merge) to use this command. After selecting a file, use this command to view the differences between different versions of the file.
Source Control
You must link Studio to an external source control management (SCM) system to use this command. Selecting this command opens a number of submenu commands involving options related to managing Studio files in a SCM system.
Find in Explorer
Use this command to navigate to the selected file in the Windows Explorer window.
