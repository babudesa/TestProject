New Text Editor
Guidewire has added simple text editor to Studio for use with XML files editable in Studio, but for which no dedicated editor exists. This text editor works with config.xml or extensions.xml, for example. This text editor is active only if you do not specify an external file editor. Guidewire recommends, however, that you link these types of files to an external editor if you plan on making complicated changes to your files.
