Security on Notes
ClaimCenter provides a set of system permissions to provide security for all notes; you may also mark a note as ‘confidential’. You may also use these permissions to define different security types for notes and assign permissions to users that relate to these ACLs. “Notes Security” in the Application Guide and “Access Control for Documents and Notes” in the Application Guide describes this in detail.
