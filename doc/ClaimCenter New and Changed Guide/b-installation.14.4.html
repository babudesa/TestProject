Support for WebLogic JNDI Data Source
You can configure ClaimCenter to use a JNDI data source managed by WebLogic. This provides a secure option for storing database credentials and allows you to manage the database connection pool through the application server. Previously, ClaimCenter could only use a JNDI data source managed by WebSphere. See “Using a JNDI Data Source” in the Installation Guide.
