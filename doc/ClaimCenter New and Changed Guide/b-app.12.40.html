PCF Input Widgets
Editable input widgets in PCF used to contain an attribute called setter that allowed arbitrary code to set the value of the field. The setter attribute is now deprecated.
The setter attribute was often used to access items in lists and fields in maps. GScript now contains a feature to replace this type of usage. PCF editable input widgets can now use value expressions that use bracket syntax to indicate a list or map value. For example, you can now use the field value myEntity.MyProperty[value]. This path expression will be used for both the getting and setting of the widget.
