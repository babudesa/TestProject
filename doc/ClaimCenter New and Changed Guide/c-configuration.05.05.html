Changes Related to Workflow
In ClaimCenter 6.0 Guidewire has made a number of changes to workflow. These include:
•
Studio provides a right-click menu command that you can use to change the step type within a workflow.
•
Studio provides the ability to create the necessary metadata for a new workflow subtype.
•
Studio opens the workflow to proper workflow step if you use the Find in path command.
See Also
•
For general information on Guidewire workflow, see “Guidewire Workflow” in the Configuration Guide.
