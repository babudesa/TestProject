New in Reporting in ClaimCenter 6.0
This topic describes what is new in Standard Reporting in ClaimCenter 6.0. These include:
•
New Claim Health Metric reports. See “Claim Health Metrics Reports” in the Reporting Guide.
•
New Sree Properties
•
New Build Properties
New Sree Properties
Guidewire has added the following properties to the sree.properties file:
•
repository.audit.enabled
•
schedule.auto.start
For more information, see “Troubleshooting Guidewire Standard Reporting” in the Reporting Guide.
New Build Properties
Guidewire has added the following property to the build.properties file:
•
build.sree_home
For more information, see “Set Up the Reporting Configuration Files” in the Reporting Guide.
