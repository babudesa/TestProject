New Compound Assignment Operators
Gosu now supports the following compound assignment operators for use in Gosu statements:
•
Add (+=)
•
Subtract (-=)
•
Divide (/=)
•
Multiply (*=)
•
Modulo (%=)
•
Bitwise AND (&=)
•
Bitwise Inclusive OR (|=)
•
Bitwise Exclusive OR (^=)
•
Bitwise Left Shift (<<=)
•
Bitwise Right Shift and Preserve Sign (>>=)
•
Bitwise Right Shift and Clear Sign (>>>=)
These operators perform an action on the contents of a variable and then set the variable to the result. For example, the += operator adds a value to the current value in a variable:
var i = 10
i += 2
print(i) // prints 12
For details and examples of each operator, see “Variable Assignment” in the Gosu Reference Guide.
