Configuring Recently Viewed Claims in the Claim Tab
You can configure recently viewed claim information in the Claim tab. You use this tab to either create a new claim, search for a specific claim, or access a recently viewed claim from a list. This topic explains how to add the loss date in the recently viewed claims for the auto loss type using Guidewire Studio.
