Message Sinks Now Unsupported
In previous releases, message sinks were deprecated in favor of new messaging plugins such as MessageTransport, MessageRequest, and MessageReply. Customers were urged to rewrite message sinks with the new messaging architecture and new messaging plugin interfaces. In this release, message sinks are fully unsupported.

Before starting the server
