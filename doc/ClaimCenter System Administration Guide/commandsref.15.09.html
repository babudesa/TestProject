ClaimCenter System Administration Guide : ClaimCenter Administrative Commands : usage_tools Command
usage_tools Command
usage_tools -help
usage_tools -password password [-server url] [-user user] [-D name=value | -count_exposures
            [-generate_csv | {-include_exposure_details | -include_monthly_details}] | 
            -start_date startdate | -end_date enddate ]
usage_tools Options
You can use any of the following options with the usage_tools command. You must always supply the -password option.
Option
Description
-count_exposures
Count the total number of exposures.
-D name=value
Sets a Java system property.
-end_date enddate
Counts the total number of exposures up to and including this date. You can only specify this option with the -count_exposures option.
-help
Displays the command usage.
-include_exposure_details
Outputs all the exposure details. You cannot use this with the -include_monthly_details option.
-include_monthly_details
Includes the exposures per month in the output. You cannot use this with the -include_monthly_details option.
-password password
Specifies the administrative password. You must specify a password.
-start_date startdate
Counts the total number of exposures from this date forward. You can only specify this option with the -count_exposures option.
-server url
Specifies the ClaimCenter host server url.
-user user
Specifies the user to run the process.
