Using the Internal Tools
WARNING Guidewire does not support the Internal Tools. Guidewire provides these tools for use during development only. Guidewire does not support the Internal Tools for production environments. Use these tools at your own risk.
The Internal Tools page is only available if the server is in development mode. You can put the server in development mode by setting the JVM parameter -Dgw.server.mode=dev. Users with the internaltools permission can then access the Internal Tools pages by pressing ALT+SHIFT+T and selecting the Internal Tools tab. The Admin role has the internaltools permission by default. For more information about server modes, see “Understanding Server Run Levels and Modes”.
