ClaimCenter System Administration Guide : Archiving Claims
Archiving Claims
WARNING Contact Guidewire Support before implementing archiving.
Archiving is the process of moving closed claims and associated data from the active ClaimCenter database into an archive database. You can still search for, retrieve and restore archived claims, but while archived, they reduce the size of the active database.
You can create a primary archive database that ClaimCenter writes to when archiving and any number of secondary archive databases. ClaimCenter does not write records to a secondary archive database, but can search a secondary archive. In a typical use case, ClaimCenter writes to a primary archive database for a certain period of time. Then, the primary archive is designated as a secondary archive for data retrieval only, and a new primary archive is specified. This prevents any one archive database from becoming too large.
