Reducing the Number of Cached Temporary Files
By default, InetSoft caches temporary files for 48 hours. Over a period of time, this can create a large number of temporary files stored on disk. To reduces the number of files, do one of the following:
•
Within the InetSoft Enterprise Manager, navigate to Performance → Cache settings and set Maximum Swap File Age to 108000000, which is 30 minutes. (30 minutes is the minimum value to which you can set this value.)
•
Open sree.properties for editing, and set the following parameter:
replet.cache.max.age=108000000
