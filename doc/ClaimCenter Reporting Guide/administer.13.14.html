Problems with Report Data
The following table lists issues that you might encounter with report data.
 
Problem
Resolution
Report returns no data
All reports return data. If one does not, it can be because there is no data available. Verify that your ClaimCenter reporting database contains data, even if it is just sample data.
Report execution appears to hang
If the data set being returned to the report is too big (15,000 or more records), the report can appear to hang. In this case, change your report criteria to reduce the number of records in the report.
