Reporting in a Clustered Environment
Guidewire requires that you configure Guidewire ClaimCenter and Guidewire Standard Reporting in a clustered configuration with a load balancer to improve speed and performance. Guidewire provides the following example for setting up a ClaimCenter cluster only as an illustration of the configuration involved. You must modify this example to meet your business needs.
