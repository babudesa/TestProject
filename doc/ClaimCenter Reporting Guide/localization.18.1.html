ClaimCenter Reporting Guide : Working with Report Development : Report Localization
Report Localization
This topic discusses how to configure ClaimCenter Standard Reporting to display a different language other than that of the base configuration reports. Guidewire ClaimCenter uses U.S. English as the default language, both in Standard Reporting and in the application itself. To display reports in a different language, you must configure ClaimCenter Standard Reporting separately from Guidewire ClaimCenter.
