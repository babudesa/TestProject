% of Reserve Change from initial reserve
Same criteria as # of Reserve Changes, percentage is calculated based on Reserve Amount changes. This is the amount of the reserves that counts in # of Reserve Changes divided by initial reserves.
Initial Reserves are defined as:
•
Any reserves created during exposure creation, or
•
After creating first approved reserve set on the claim, any reserves created within the InitialReserveAllowedPeriod are considered initial reserved.
Also, the configuration parameter InitialReserveAllowedPeriod, which defines the number of days after first initial reserve, is created. This means that all reserve changes within that period count as initial reserves.
Deferred: % of Reserve Change from initial user set reserve
(Current Incurred net- Initial user-set reserve) / Initial user-set reserve
