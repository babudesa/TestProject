Waiving Deductibles
A deductible can be waived if it has not been applied to any payment. This is done in the Exposures Details screen, where the deductible's Waived flag can be set to Yes. This field is not editable if the deductible has already been paid (although if something causes it to become unpaid, then the waived field is again editable). If you waive a deductible, then the Apply Deductible button does not appear in the check wizard after selecting related exposures. You must also have the correct permission of Edit Deductible.
Note: Waiving a deductible is usually done by more experienced adjusters. They are often waived in no-fault states if the insured is not at fault.
1.
Navigate to the Exposures Details screen and click Edit.
2.
Ensure that a coverage is selected.
3.
In the Deductibles section, select the Waive Deductible radio button.
4.
Enter an edit reason.
5.
Update your work.
