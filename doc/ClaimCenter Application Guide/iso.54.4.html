ISO Permissions
Who can work with ISO?
If you have the permissions to view the claim/exposure, then you can view the ISO match reports. This also includes editing the claim/exposure or clicking the Send to ISO button so that the message is sent to ISO.
However, the Administer integration permission (code = integadmin), can be appended to a role such as a supervisor or adjuster. Use this permission to see and edit information that is not of interest to most users but can help in rare cases. For example, you can edit information after the ISO state of the exposure/claim has become out of synch with the actual ISO server.
