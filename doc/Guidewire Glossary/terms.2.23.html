XML
Extensible Markup Language (XML) is a standard that describes complex structured data in a text-based format with strict syntax for easy data interchange. For more information, refer to the World Wide Web Consortium at http://www.w3.org/XML.
XPath
XPath (XML path language) is a simple syntax for describing an object or property from a root object. Variants of XPath syntax are used to integrate within the web service IDataObjectAPI interface. For example, ClaimCenter uses XPaths in ISO integration to build a hierarchy of fields to send to ISO. See also field path.
