ClaimCenter Installation Guide : Installing a ClaimCenter Development Environment : Using SQL Server or Oracle in a Development Environment
Using SQL Server or Oracle in a Development Environment
ClaimCenter supports SQL Server and Oracle databases, as described in “Configuring the Database”. You can use Oracle or SQL Server instead of the QuickStart database. Such a configuration can be used for testing. Guidewire does not support this configuration for a production environment.
For instructions on creating a SQL Server or Oracle database instance, consult “Configuring the Database”. For instructions on configuring ClaimCenter to connect to the database, consult “Configuring a Database Connection”. After these configuration settings are complete, use the commands listed in “QuickStart Command Tools” to control the application server.
