ClaimCenter Installation Guide : Installing ClaimCenter
Installing ClaimCenter
Installing ClaimCenter is a multi-step process that requires you to perform several procedures. The initial installation process can take from two hours to a full day. This topic begins with an overview of the installation process and then guides you through each procedure culminating with the deployment of ClaimCenter to a production environment.
For QuickStart installation instructions, see "QuickStart" instead of this topic.
Note: The examples in this topic are based on a Windows environment. With the exception of the path specification format and some example paths, the commands and locations documented are the same as in a UNIX environment.
