ClaimCenter Installation Guide : Introduction to Installation
Introduction to Installation
This guide describes how to install ClaimCenter.
This topic describes the installation options that are available and provides guidance on selecting which option to choose.
The ClaimCenter Installation Guide includes:
•
An overview that discusses the different ways to install ClaimCenter that starts with this topic.
•
“Preparing a ClaimCenter Environment” discusses supported database and application server platforms and provides steps to prepare a development or production environment for ClaimCenter.
•
“Installing a ClaimCenter Development Environment” explains how to install a ClaimCenter development environment using the QuickStart server and database or Tomcat.
•
“Installing a ClaimCenter Production Environment” explains how to deploy ClaimCenter to an application server and database server production environment.
•
“Commands Reference” lists and describes the QuickStart and build commands. Other ClaimCenter command line utilities are described in “ClaimCenter Administrative Commands” in the System Administration Guide.
